Question-1: What are the different ways to select an element in the DOM?

There are multiple way to select an element using DOM, such as: 1. document.getElementById() 2. document.getElementsByClassName() 3. document.getElementsByTagName() 4.1 document.querySelector() 4.2 document.querySelectorAll() 5. document.getElementsByName()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Difference between innerHTML, innerText, and textContent: innerHTML: we can get or we can set HTML directly using innerHTML. innerText: Retrieves or sets visible text inside an element. textContent: Retrieves or sets all text content, including hidden text. diffrence: innerHTML is slower, innerText is faster and it's ignore hidden content and textContent is best for performance.

Question-3: What is event delegation in the DOM?

In javascript event delegation is a technic or concept where we set the event to the parent instead of child adn this leverages the bubbling phase of event propagation to handle events for child elements from the parent.

Question-4: What is event bubbling in the DOM?

Event bubbling is a mechanism of DOM. Where an event start from the target and it bubbles up to its parent or grandParents which is root of the documents.

Question-5: How do you create, add, and remove elements using JavaScript?

To create element we can use "document.createElement()" and then to add the created element we can use "appendChild()" finally to remove an element we canuse "remove()"
